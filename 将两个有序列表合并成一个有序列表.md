## 尾递归 ##

如果要返回递归的次数 需要使用一个全局变量 a = [0]    然后每递归一个就 a[0] +=1 最终返回的就是一个最终的次数


尾递归方法一

	a = [1,3,5,8,9]
	b = [2,4,6]
	
	def sort1(a,b):
	    c = []
	    if len(a) == 0 :
	        c.extend(b)
	        return c
	    elif len(b) == 0:
	        c.extend(a)
	        return c
	    else:
	        if a[0] < b[0]:
	            c.append(a[0])
	            del a[0]
	        else:
	            c.append(b[0])
	            del b[0]
	        c += sort1(a, b)
